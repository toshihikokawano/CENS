Configuration

Configuration File

Athough CENS does not require any input data for general use, some internal variables can be modified by a config.dat file in the current directory. This is a simple ASCII file, containing Keyword = Value lines. The keywords are 

MaxDiscreteLevels = 10000
MaxGammaLines     = 100
ENSDFDirectory = /usr/local/share/ENSDF/adopted
RIPLDirectory = /usr/local/share/coh/levels
EnergyUnit = MeV

MaxDiscreteLevels is the max number of discrete levels to be loaded from ENSDF, and MaxGammaLines is the max number of gamma decay from each discrete level. The default values for these variables are defined in cens.h, and probably no need to change them.

ENSDFDirectory and RIPLDirectory are default directories where CENS finds ENSDF and RIPL data files. The default value is the current directory. The RIPL files are used to extract Internal Conversion Coefficients (ICC) that are not given in ENSDF. If any RIPL files are found, ICC will be zero, unless ENSDF has its value.

EnergyUnit controls the unit of output energy values. The acceptable values are eV, keV, and MeV, and the default unit is MeV.
