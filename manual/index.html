Introduction

General Information

The CENS code translates adopted discrete levels in ENSDF into a Hauser-Feshbach code readable format, such as the discrete level section in RIPL (Reference Input Parameter Library). Missing information, like unknown spin and parity, gamma decay branching ratio, etc. is estimated by considering the statistical properties of nuclear strucure. CENS also prints discrete level information in the XML format.

History

CENS development started in Nov. 2016 (ver.0.0). Although the main part of ENSDF parser was done at that time, it has been put aside for a long time. In Mar. 2021, two functions were added; (i) statistical analysis of discrete levels (density and spin distribution), and (ii) reading Internal Conversion Coefficients from RIPL when ENSDF does not have this information. This is the first functional version (0.1).

In May 2021, CENS was released as open-source software from Los Alamos National Laboratory.
